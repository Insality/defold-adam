AdamInstance:initialize(initial_state, transitions, variables)
Adam Instance constructor function.

Parameters:

  • initial_state StateInstance The initial FSM state. It will be triggered on start
  • transitions StateInstance[] The array of next structure: {state_instance, state_instance, [event]}, describe transitiom from first state to second on event. By default event is adam.FINISHED
  • variables table Defined FSM variables. All variables should be defined before use

Returns:

    AdamInstance
AdamInstance:start()
Start the Adam Instance. On create FSM is not started and should be started manually. You can use `local adam = AdamInstance({}):start()`

Returns:

    AdamInstance

AdamInstance:final()
Adam final function. Place in script/gui_script final function on when you wanna to stop FSM. Important function, since it track global list of Adam instances

actions.fsm Functions

actions.fsm.send_event(target, event_name[, delay[, is_every_frame]])
Send event to target DMaker instance

Parameters:

  • target string or dmaker The target instance for send event. If there are several instances with equal ID, event will be delivered to all of them.
  • event_name string The event to send
  • delay number Time delay in seconds (optional)
  • is_every_frame bool Repeat every frame (optional)
actions.fsm.broadcast_event(event_name[, is_exclude_self[, delay]])
Broadcast event to all active FSM

Parameters:

  • event_name string The event to send
  • is_exclude_self bool Don't send the event to self (optional)
  • delay number Time delay in seconds (optional)

actions.math Functions

actions.math.add(source, value[, is_every_frame[, is_every_second]])
Add operator

Parameters:

  • source string The variable name from DMaker instance
  • value number or variable The number or Dmaker variable to add
  • is_every_frame boolean Check true, if action should be called every frame (optional)
  • is_every_second boolean Check true, if action should be called every second. Initial call is not skipped (optional)

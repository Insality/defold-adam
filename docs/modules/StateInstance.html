Module StateInstance

User to describe the states for your FSM.

Contain actions what will be triggered, on enter to this state.

Functions

event(event_name) Trigger event to FSM.
set_debug(state) Set debug state of state.

Functions

event(event_name)
Trigger event to FSM. If state changed by one of Action, other actions below of it will not executed

Parameters:

  • event_name string The event to send
set_debug(state)
Set debug state of state. If true, will print debug info to console

Parameters:

  • state boolean The debug state

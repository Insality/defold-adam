Module AdamInstance

All Adam instances created by this calss.

Instantiate it via `adam.new()`.

Functions

initialize(initial_state, transitions, variables)
Adam Instance constructor function.

Parameters:

  • initial_state StateInstance The initial FSM state. It will be triggered on start
  • transitions StateInstance[] The array of next structure: {state_instance, state_instance, [event]}, describe transitiom from first state to second on event. By default event is adam.FINISHED
  • variables table Defined FSM variables. All variables should be defined before use

Returns:

    AdamInstance
